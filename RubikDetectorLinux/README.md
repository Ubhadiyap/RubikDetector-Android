## Build requirements
   - [Cmake 3.4.1 or newer](https://cmake.org/download/)
   - [OpenCV 3.2.0](https://github.com/opencv/opencv/releases/tag/3.2.0) installed on the local machine
   
To build:
```
mkdir build
cd build
cmake ../
```
Then just run it with:
```
./rubikdetector_desktop
```
That's it!
